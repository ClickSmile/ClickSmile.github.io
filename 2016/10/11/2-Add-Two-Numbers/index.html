2. Add Two Numbers

Posted on 2016-10-11   |   In LeetCode

Problem:

You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

Solution:

Just add two values of these list. What you need to remember is the carry position to update each time. Use while loop to traversal each node.
